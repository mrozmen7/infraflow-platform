# InfraFlow Development Documentation

This folder contains the detailed development notes, module walkthroughs and learning
evidence created while building InfraFlow.

The repository root README presents InfraFlow as an enterprise reference system. This
folder keeps the step-by-step engineering documentation available without making the
project front page look like a course outline.
